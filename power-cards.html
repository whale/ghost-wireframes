Power through simplicity

Open source

Ghost is public code. MIT licensed, built on Node.js, maintained since 2013. Not a black box you're renting.

Self-hostable

Download Ghost and run it on your own server. Ghost(Pro) is the managed option, but you're never locked in.

API-first

Content API and Admin API. Build headless sites, power mobile apps, automate publishing workflows.

Open protocols

RSS, ActivityPub, oEmbed, structured data. Ghost speaks the open web, not a walled garden.

Fully portable

Import from Substack, WordPress, Medium, and more. Export everything as JSON anytime. Your data is yours.

Extensible

Webhooks, Zapier, custom integrations with API credentials. Connect Ghost to 8,000+ apps or build your own.
